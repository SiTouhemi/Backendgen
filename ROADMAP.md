# Roadmap

## 0.2 alpha release gate

- Keep the specification, IR, generation, regeneration, CLI, and MCP suites green.
- Execute all generated projects and critical CRUD/auth/tenant/reservation behavior against PostgreSQL in CI.
- Validate the benchmark methodology with design partners; do not publish synthetic savings as model results.
- Complete an external security and generated-code review.

## After design-partner validation

- Stabilize adapter and feature-pack APIs.
- Improve generated dependency locking and upgrade automation.
- Consider a second target only after NestJS/PostgreSQL conformance is stable.
- Decide whether optional hosted collaboration adds value. Local generation remains supported.

FastAPI, a visual editor, a public marketplace, and SaaS billing are not committed roadmap items.
